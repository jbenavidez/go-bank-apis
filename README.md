# go-bank-apis(WIP)


APIS set that allow users to create customers and accounts


# Customers APIs
<ul>
<li>Get All customers:
    <ul>
    <li>Endpoint: {domain-name}/customers</li>
     <li>Method: GET</li>
    </ul>
</li>
<li>Get customer:
    <ul>
    <li>Endpoint: {domain-name}/customers/{customer-id}</li>
     <li>Method: GET</li>
    </ul>
</li>
<li>Create customer:
    <ul>
    <li>Endpoint: {domain-name}/customers</li>
     <li>Method: Post</li>
    </ul>
</li>
<li>Update customer:
    <ul>
    <li>Endpoint: {domain-name}/customers/{customer-id}</li>
     <li>Method: PUT</li>
     </ul>
</li>
</ul>
